Export & sync to the live store

The storefront reads products.json every time it loads. Replace that one file on your hosting and every visitor sees the update — no other changes needed.

How this actually works

  1. Make your changes here (add, edit, delete products).
  2. Click Download products.json below.
  3. Upload that file to your web host, overwriting the existing products.json — same folder as index.html.
  4. Done. The store fetches this file fresh on every page load, so it updates immediately for every visitor, on any device.

📸 Adding product photos

Every product already has a predictable image slot waiting — the storefront shows a "Photo coming soon" placeholder until the matching file exists.
  1. Create a folder named images/products/ next to index.html on your host.
  2. For each product, save its photo using the exact filename shown in the Image File column on the Manage Catalogue tab (or under the product name when adding a new one).
  3. Use JPG, roughly square, under 300KB each works best for load speed.
  4. Upload the photos into that folder. No code changes, no re-export needed — the site loads them directly by filename the next time anyone visits.
